# _Crunchyroll Website Clone_

#### _A website made to imitate crunchyroll.com as of 2/15/1029_

#### By _**Aurora Shido-Wagenet**_

## Description

_This application was done as a Sass and CSS study._

## Specs

Specification |
------------- |
Visually match the layout of Crunchyroll Website |
Visually match the content of the webpage |
Adjust media queries to display more appropriately on different devices |

## Setup/Installation Requirements

* _Use your command terminal (gitbash if you're using a windows machine) to clone the latest commit from Github, using the url provided_
* _Open with a text editor such as Atom to view the code_
* _Then, open the file labeled "index.html" with your preferred web browser, such as Chrome, to view the actual page_
